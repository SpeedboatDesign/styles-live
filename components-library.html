Interaction state matrix

Every interactive primitive must support all states below before shipping.

Default

Clear affordance, standard contrast.

Hover

Small visual shift, no layout movement.

Focus-visible

2px focus ring with AA contrast minimum.

Active

Pressed depth + stronger tonal response.

Disabled

Lower contrast + blocked pointer interaction.

Loading

Retain dimensions; show progress affordance.

Error

Use clear guidance and a recovery action.

Success

Confirm completion and suggest next step.

Responsive behavior rules

375–429 · Mobile

Single column. Bottom navigation preferred. Compact spacing scale.

430–767 · Large mobile

Single column with denser utility rows and larger tap zones.

768–1023 · Tablet

Two-column content, split hero stacks, expanded navigation labels.

1024–1279 · Laptop

Introduce side index / persistent contextual navigation.

1280+ · Desktop

Use full rhythm spacing and complex multi-column modules.

Accessibility release checklist

  • Text contrast passes WCAG AA in every state.
  • Interactive controls remain fully keyboard operable.
  • Focus ring is visible at 200% zoom and in high glare conditions.
  • Touch targets are at least 44×44px.
  • Reduced-motion users get non-animated alternatives.
  • Error states explain what happened and how to recover.
